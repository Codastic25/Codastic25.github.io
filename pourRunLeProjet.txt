# Workflow pour mettre à jour le projet React + Vite + GitHub Pages

## 1. Travailler en local
- Modifier les fichiers dans `src/` (JSX, CSS, etc.)
- Vérifier les changements avec :
  npm run dev
- Voir le site localement sur http://localhost:5173/

## 2. Préparer la production
- Compiler le projet :
  npm run build

- Copier le contenu du build vers la racine pour GitHub Pages (user site) :
  cp -r dist/* .

## 3. Publier sur GitHub Pages
- Ajouter et commit les fichiers build :
  git add .
  git commit -m "Deploy latest build"
  git push

## Ne pas oublier de remettre le index.html local avec les chemins src pour le npm run dev :

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>site-web-codastic25</title>
    <script type="module" src="/src/main.jsx"></script>
    <link rel="stylesheet" href="/src/App.css" />
  </head>
  <body>
    <div id="root"></div>
  </body>
</html>
